# IES

存储介质消除一体机管理界面

# react-umi-electron

# Install dependencies

* $ yarn
* $ cd app
* $ yarn 

# start the dev server

* $ yarn electron-dev

# Build dist

* $ yarn build  首先在根目录下打包umi生成build文件
* $ cd app
* $ yarn build:linux linux系统下打linux环境下运行的包
* $ yarn build:win windows系统下打windows环境下运行的包
